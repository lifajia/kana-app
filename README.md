### 日语五十音学习
在线学习日语五十音，包含基础五十音、浊音、拗音的闪卡学习和平假名片假名练习。支持语音发音、进度追踪和错题集功能。
基于Nuxt,Vue3构建的本地SSG应用,方便SEO,无任何接口交互.

在线体验:[日语五十音学习](https://lifajia.github.io/kana-app/)
